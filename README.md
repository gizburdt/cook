Opinionated package to speed up new Laravel projects.

## Installation

You can install the package via composer:

``` bash
composer require gizburdt/cook --dev
```

## Usage

``` php
php artisan cook:publish

php artisan cook:install
```

## Testing

``` bash
composer test
```
